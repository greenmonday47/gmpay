API Documentation

Get User Information

Endpoint: /api/user/{id}

Method: GET

Description: Returns information about a specific user with the given ID.

      Request: GET /api/user/123
      Response:
      {
        "id": 123,
        "name": "John Doe",
        "email": "johndoe@example.com"
      }

Update User Information

Endpoint: /api/user/{id}

Method: PUT

Description: Updates information about a specific user with the given ID.

      Request: PUT /api/user/123
      Body:
      {
        "name": "Jane Doe",
        "email": "janedoe@example.com"
      }
      Response:
      {
        "message": "User updated successfully."
      }
