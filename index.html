API Documentation

Charge User

Endpoint: https://api.gmpayapp.com/api/v2/transactions/process

Method: POST

Description: You will use the endpoint to debit GMPay accounts.

      Request: POST https://api.gmpayapp.com/api/v2/transactions/process
      Request Body:
      {
    "service": "Checkout",
    "amount": (Number Required),
    "account": (String Required | Must start with country code or actual number without leading 0),
    "pubKey": (String Required | merchant public key)
      }
      Response:
      {
   "status": (HTTP STATUS CODE),
   "statusText": (HTTP STATUS NAME),
   "data" : {
     "reference" : (TRANSACTION ID),
     "message": (HUMAN READABLE MESSAGE),
     "amount": (ORIGINAL AMOUNT),
     "username": (USER FULLNAME),
     "phoneNumber": (USER PHONE NUMBER),
        }
      }

Update User Information

Endpoint: /api/user/{id}

Method: PUT

Description: Updates information about a specific user with the given ID.

      Request: PUT /api/user/123
      Body:
      {
        "name": "Jane Doe",
        "email": "janedoe@example.com"
      }
      Response:
      {
        "message": "User updated successfully."
      }
